# 반려동물 오늘의 운세 — Design Spec

**Date:** 2026-03-26
**Status:** Approved

---

## Overview

반려동물의 이름과 생년월일을 입력하면 오늘의 기분 운세를 귀엽고 사주 스타일로 보여주는 모바일 웹앱. 결과 카드는 인스타그램 스토리 캡처에 최적화.

---

## Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Language:** TypeScript
- **Rendering:** Client-side only (no backend, no DB)
- **Deployment target:** Vercel (optional)

## Pages & Routes

## Fortune Calculation — Multi-Factor Saju Style

입력값과 오늘 날짜를 조합해 순수 결정론적으로 계산. 서버 없이 클라이언트에서 처리.

### Factor 1: 12지지 (Zodiac) — 태어난 해 기준
- 출생 연도 % 12 → 12지지 매핑 (자/축/인/묘/진/사/오/미/신/유/술/해)
- 성격 키워드, 기본 에너지 레벨 결정

### Factor 2: 오행 (五行) — 태어난 월 기준
- 월 → 오행 매핑 (1-2월: 木, 3-4월: 火, 5-6월: 土, 7-8월: 金, 9-10월: 水, 11-12월: 木)
- 오늘 날짜의 오행(날짜 % 5)과 상성 계산 → 오늘 기분 수치 (좋음/보통/최고)

### Factor 3: 오늘 에너지 — 요일 + 날짜
- 요일(0-6) + 오늘 날짜(1-31) 조합 → 행운 카테고리 인덱스
- 행운의 아이템/간식/장소 선택

### Factor 4: 생일 근접도
- 오늘과 생일(월/일)의 차이가 7일 이내 → 특별 생일 메시지 트리거

### Output
- **오늘의 기분:** 5단계 (최고예요 / 신남 / 보통 / 살짝 졸림 / 삐짐 모드)
- **기분 이유:** 짧은 귀여운 설명 한 줄
- **행운의 아이템:** 간식/장난감/장소 중 하나
- **오늘의 한 마디:** 주인에게 보내는 귀여운 메시지 (인스타 캡처용 하이라이트)

---

## Message Pool

- 기분 카테고리별 메시지 5개 이상
- 행운 아이템 카테고리별 5개 이상
- 한 마디 메시지 20개 이상 (예: "오늘은 간식을 3개 주면 주인을 더 사랑하게 됩니다!", "오늘의 행운의 산책 코스는 공원!")

## UI/UX

### 입력 화면 (`/`)
- 풀스크린 모바일 레이아웃
- 반려동물 이름 텍스트 입력
- 생년월일 date picker
- "오늘 운세 보기" 버튼
- 파스텔 배경, 귀여운 폰트

### 결과 카드 (`/result`)
- **9:16 비율 카드** — 스크린샷 최적화
- 상단: 반려동물 이름 + 오늘 날짜
- 중앙: 기분 이모지 + 기분 레벨
- 하단: 행운 아이템 + 오늘의 한 마디
- Web Share API 공유 버튼
- "다시 보기" 버튼 → 입력 화면으로
